# AutoPager 自动翻页

一个 Chrome 扩展(Manifest V3),在你向下滚动时**自动加载下一页并无缝拼接到当前页面**,实现"无限滚动"式的连续浏览。适用于论坛列表、搜索结果、博客归档、图集等分页内容。

灵感来自经典的 AutoPagerize / uAutoPagerize,规则库兼容,同时内置通用识别与精致的中文设置界面。

## ✨ 功能特性

- **规则库 + 通用识别双引擎**
  - 优先使用 AutoPagerize SITEINFO 规则库(内置每日自动更新,约 3800+ 条规则)精确匹配。
  - 未命中规则时,自动回退到通用启发式识别:`rel="next"`、常见分页容器/类名、以及"下一页 / next / more"等链接文本,并内置对 Discuz! 等常见论坛结构(`#threadlisttableid`、`a.nxt` 等)的支持。
- **精致的分页指示条**:每加载一页,在内容之间插入一条与列表**整行宽度对齐**的横条(加载中带旋转动画,完成显示"第 N 页"),自动适配表格 / 普通容器两种结构。
- **按需加载,不过度预加载**:加载一页后需要你**再次滚动**才会加载下一页,不会一口气把后面几十页全拉下来;可选的预读(prefetch)也仅提前 1 页。
- **旧编码支持**:后台抓取子页面时会**检测真实字符集**(HTTP 头 / `<meta charset>`),正确解码 GBK、Big5、Shift_JIS 等非 UTF-8 页面,避免中文乱码。
- **页内悬浮控制**:右上角悬浮按钮(Shadow DOM 隔离,不污染宿主样式),可暂停/继续、立即加载下一页、打开设置;支持常显 / 隐藏 / 悬停显示,透明度与大小可调。
- **灵活的排除规则**:按行配置通配符(`*`)或 `/正则/` 排除不想翻页的页面。
- **右键菜单**:立即加载下一页、暂停/继续、排除当前页、打开设置。
- **其他**:触发距离可调、新标签页打开链接、标记已加载过的链接、修复滚动穿透等。
- **全中文界面**:设置页采用侧栏 + 卡片布局、iOS 风格开关与分段控件,并带滚动高亮导航。

## 📦 安装(开发者模式加载)

1. 下载或克隆本仓库:
   ```bash
   git clone git@github.com:Faceisme/auto-pager.git
   ```
2. 打开 Chrome,访问 `chrome://extensions`。
3. 打开右上角的 **开发者模式**。
4. 点击 **加载已解压的扩展程序**,选择本仓库文件夹。
5. 修改代码后,回到该页面点击扩展卡片上的 **刷新 ↻** 使改动生效。

## 🚀 使用

安装后默认在所有页面启用。打开一个分页内容页面(如论坛版块列表),向下滚动到接近底部,扩展会自动加载并拼接下一页。

- 点击工具栏图标可查看当前页状态、快速开关、排除本页或打开设置。
- 点击右上角悬浮按钮可暂停/继续或立即加载下一页。

## ⚙️ 设置说明

| 分区 | 说明 |
| --- | --- |
| 排除页面 | 配置不启用自动翻页的地址(通配符或正则) |
| 行为 | 触发距离、是否新标签页打开、预读、修复滚动穿透、单篇内容抑制翻页 |
| 界面 | 悬浮按钮显示方式 / 透明度 / 大小、分页指示条、右键菜单、标记已加载链接 |
| 规则库 | 手动更新 SITEINFO 规则库、查看规则数量与更新时间 |
| 数据 | 导出 / 导入设置、恢复默认 |

## 🗂 项目结构

```
auto-pager/
├── manifest.json     # MV3 清单
├── background.js     # Service Worker:跨域抓取、规则库更新、右键菜单
├── content.js        # 内容脚本:检测、拼接、滚动加载、页内 UI
├── content.css       # 页内指示条与标记样式
├── common.js         # 内容脚本 / 设置页 / 弹窗共享的工具函数
├── options.html/js/css  # 设置页
└── popup.html/js     # 工具栏弹窗
```

## 📝 说明

- 规则库来源为社区维护的 AutoPagerize SITEINFO 镜像;wedata.net 已停服,故改用 `tophf/autopagerize` 镜像。
- 本扩展在页面的隔离环境中运行,拼接内容前会将相对链接改写为绝对地址。
